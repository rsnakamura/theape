The Event Timer¶

The EventTimer Class¶

EventTimer([seconds, event]) A timer object to set an event
EventTimer.event Threading event for the timer to set
EventTimer.set_event() Sets the event
EventTimer.start() The main interface - clears the event then starts the timer
EventTimer.clear() A convenience method for users to call the event.clear method.
EventTimer.wait([timeout]) Calls event.wait if timeout is None, uses self.seconds
EventTimer.close() Cancels the timer and sets the event.
EventTimer.is_set()
return:True if self.event is set

The Wait (decorator) Function¶

wait(method) Decorator to wait for previous timers and to start a new one on exit

The wait decorator expects that it is decorating a method in a class that has an EventTimer object as a timer property (it references self.timer).
