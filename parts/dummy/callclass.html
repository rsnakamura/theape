Call Class¶

The call class is a primitive class for dummies to use in __getattr__ to log un-implemented calls. Debugging use only.

CallClass¶

CallClass([message, payload]) A class for dummies to return when called.
CallClass.__call__(*args, **kwargs) returns the payload after logging items
CallClass.__name__

A class to fake calls and properties.
