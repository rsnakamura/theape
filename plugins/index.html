Plugins¶

This is a folder to contain plugins. Plugins are meant to be the code available for the user to pick (via the configuration file). This differentiates them from parts which are available to plugin-creators to use but not meant to be used directly by the end user (if a part is meant to be used stand-alone, a plugin front-end should be built for it).
